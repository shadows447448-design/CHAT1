# 江湖爬塔（wuxia_spire）安装与使用指南

`wuxia_spire.py` 是一个单文件、中国风、类《杀戮尖塔》的命令行爬塔卡牌小游戏。它不依赖图形界面，适合在本地终端快速体验，也支持固定随机种子进行自动演示和测试。

## 一、环境要求

- Python 3.9+（推荐 Python 3.11 或更新版本）
- 可运行命令行终端：macOS/Linux Terminal、Windows PowerShell、Windows Terminal 均可
- 游戏本体只使用 Python 标准库；运行测试需要安装 `requirements.txt` 中的测试依赖

## 二、安装步骤

### 1. 获取项目代码

如果你已经在本仓库目录中，可以跳过本步。否则先克隆仓库并进入目录：

```bash
git clone <your-repo-url>
cd CHAT1
```

### 2. 创建虚拟环境（推荐）

macOS / Linux：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Windows PowerShell：

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3. 安装依赖

只玩游戏不需要额外第三方包；如需运行测试，请安装仓库依赖：

```bash
pip install -r requirements.txt
```

## 三、快速开始

### 交互模式（推荐新手）

```bash
python wuxia_spire.py
```

进入后按提示操作：

1. 选择难度：`1` 简单、`2` 标准、`3` 困难；直接回车默认简单。
2. 战斗胜利后遇到卡牌奖励时，输入 `1`、`2` 或 `3` 选择一张卡；直接回车默认第 1 张。
3. 游戏结束后查看层数、生命、金币、卡组大小、遗物和本局日志。

### 自动演示模式

无需输入，适合快速查看游戏流程：

```bash
python wuxia_spire.py --auto
```

指定难度、随机种子和层数：

```bash
python wuxia_spire.py --auto --difficulty normal --seed 42 --floors 10
```

查看命令帮助：

```bash
python wuxia_spire.py --help
```

## 四、玩法说明

- **目标**：从第 1 层爬到 Boss 层，尽量保持生命并构筑更强卡组。
- **卡牌类型**：
  - 攻击牌：造成直接伤害，例如 `直刺`、`青龙摆尾`。
  - 技能牌：获得格挡、抽牌或增加能量，例如 `护体`、`丹田吐纳`、`聚气`。
- **能量**：每回合默认 3 点，打出卡牌会消耗对应能量。
- **格挡**：抵消敌人本回合攻击伤害，每回合开始重置。
- **遗物**：关键楼层获得被动能力，例如每回合加格挡或胜利后加金币。
- **奇遇**：第 3、6、9 层会触发事件，可能回血、加最大生命或改变金币。
- **难度**：
  - 简单：推荐新手，玩家初始生命更高，敌人较弱。
  - 标准：默认平衡体验。
  - 困难：敌人更强，但战斗胜利金币略高。

## 五、新手建议

1. 第一次建议选择 `1 简单`，熟悉生命、能量和格挡机制。
2. 看到低费卡时优先考虑，避免手里有牌却没有能量打出。
3. 敌人将要攻击时，格挡牌价值更高；`破甲掌` 也会因为敌人攻击意图获得额外伤害。
4. 如果获得 `温玉佩`，防御压力会明显降低；如果获得 `铜钱剑穗`，后续金币会更充足。
5. 自动模式可以用同一个 `--seed` 复盘同一局，便于比较不同选择。

## 六、测试与验证

运行全量测试：

```bash
pytest -q
```

只运行游戏相关测试：

```bash
pytest -q tests/test_wuxia_spire.py
```
