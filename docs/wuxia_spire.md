# 江湖爬塔（wuxia_spire）安装与使用指南

`wuxia_spire_web/` 是一个中国风、类《杀戮尖塔》的可视化鼠标卡牌爬塔小游戏。玩家不需要输入文字命令操作游戏：点击地图节点前进，点击或拖拽卡牌出招，点击奖励卡完成构筑。

仓库中仍保留 `wuxia_spire.py`，用于自动演示、固定随机种子复盘和自动化测试。

## 一、环境要求

- Python 3.9+（推荐 Python 3.11 或更新版本）
- 现代浏览器：Chrome、Edge、Firefox、Safari 均可
- 游戏本体只使用 HTML/CSS/JavaScript 和 Python 标准库；运行测试需要安装仓库依赖

## 二、安装步骤

### 1. 获取项目代码

如果你已经在本仓库目录中，可以跳过本步。否则先克隆仓库并进入目录：

```bash
git clone <your-repo-url>
cd CHAT1
```

### 2. 创建虚拟环境（推荐）

macOS / Linux：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Windows PowerShell：

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3. 安装测试依赖

只玩鼠标版游戏不需要额外第三方包；如需运行测试，请安装仓库依赖：

```bash
pip install -r requirements.txt
```

## 三、鼠标版快速开始（推荐）

在项目根目录启动静态文件服务器：

```bash
python -m http.server 8000
```

用浏览器打开：

```text
http://localhost:8000/wuxia_spire_web/
```

进入游戏后只需要鼠标操作：

1. **开始页**：点击难度按钮，再点击“开始闯塔”。
2. **地图页**：点击发光的下一层节点前进。
3. **战斗页**：
   - 点击攻击牌，或把攻击牌拖到敌人区域。
   - 点击防御/技能牌，或把防御牌拖到少侠区域。
   - 点击“结束回合”让敌人行动。
4. **奖励页**：点击 3 张奖励牌之一加入卡组，也可以点击“跳过卡牌”。
5. **奇遇/结算页**：点击按钮继续或重新开始。

## 四、Python 自动演示模式

无需交互，适合快速查看流程或复现测试：

```bash
python wuxia_spire.py --auto
```

指定难度、随机种子和层数：

```bash
python wuxia_spire.py --auto --difficulty normal --seed 42 --floors 10
```

查看命令帮助：

```bash
python wuxia_spire.py --help
```

## 五、玩法说明

- **目标**：从第 1 层爬到 Boss 层，尽量保持生命并构筑更强卡组。
- **地图节点**：战斗、精英、奇遇和 Boss 都在地图上以节点呈现，当前可进入节点会高亮。
- **卡牌类型**：
  - 攻击牌：造成直接伤害，例如 `直刺`、`青龙摆尾`。
  - 技能牌：获得格挡、抽牌或增加能量，例如 `护体`、`丹田吐纳`、`聚气`。
- **能量/真气**：每回合默认 3 点，打出卡牌会消耗对应真气。
- **格挡**：抵消敌人本回合攻击伤害，每回合开始重置。
- **遗物**：关键楼层获得被动能力，例如每回合加格挡或胜利后加金币。
- **奇遇**：部分楼层会触发事件，可能回血、加最大生命或改变金币。
- **难度**：
  - 简单：推荐新手，玩家初始生命更高，敌人较弱。
  - 标准：默认平衡体验。
  - 困难：敌人更强，但战斗胜利金币略高。

## 六、新手建议

1. 第一次建议选择“简单”，熟悉生命、真气和格挡机制。
2. 低费牌更稳定，能减少“手里有牌却打不出”的情况。
3. 敌人显示攻击意图时，优先考虑格挡牌；`破甲掌` 也会因为敌人攻击意图获得额外伤害。
4. 攻击牌拖到敌人、防御牌拖到少侠身上，能更直观地区分出牌目标。
5. 自动模式可以用同一个 `--seed` 复盘同一局，便于比较不同选择。

## 七、测试与验证

运行全量测试：

```bash
pytest -q
```

只运行游戏相关测试：

```bash
pytest -q tests/test_wuxia_spire.py
```

检查前端脚本语法：

```bash
node --check wuxia_spire_web/game.js
```
